# Argus API Specification — v0.1.0 (D1 baseline)

- **Status:** Active; D1 endpoints locked by `tests/test_api_contracts.py`
- **Last updated:** 2026-06-27
- **Base URL (dev):** `http://localhost:8000`
- **Base URL (prod):** Cloud Run custom domain (see ADR-0008)
- **OpenAPI spec:** `GET /openapi.json` (auto-generated by FastAPI)
- **Interactive docs:** `GET /docs` (Swagger UI, dev only)

---

## Conventions

| Convention | Value |
|---|---|
| Content-Type | `application/json` |
| Timestamps | ISO-8601 UTC strings (e.g. `"2024-02-08T00:00:00+00:00"`) |
| Coordinates | GeoJSON (lon, lat order; WGS-84) |
| Evidence classes | `"measured"` · `"modeled"` · `"inferred"` (INV-3) |
| Attribution | Responses containing Open-Meteo data carry `"_attribution"` (CC BY 4.0) |
| Error body | `{"detail": "<human-readable message>"}` |

## Endpoints

### `GET /health`

Liveness probe. Always returns 200 if the server process is running.

**Response 200:**
```json
{
  "status": "ok",
  "version": "0.1.0"
}
```

---

### `GET /aois`

List all configured Areas of Interest.

**Response 200:**
```json
{
  "items": [
    {
      "id": "tobago",
      "name": "Tobago AOI",
      "geometry": { "type": "Polygon", "coordinates": [[...]] },
      "domains": ["marine_oil"],
      "active": true
    }
  ],
  "count": 1
}
```

---

### `GET /aois/{aoi_id}`

Return a single AOI by ID.

**Path params:** `aoi_id` (string) — AOI slug (e.g. `"tobago"`)

**Response 200:** `AOISchema` (see above item shape)

**Response 404:**
```json
{"detail": "AOI 'unknown' not found"}
```

---

### `GET /aois/{aoi_id}/observations`

List Observations for an AOI. Observations are linked via their parent `AnalysisRun`.

**Query params:**
- `status` (optional) — filter by status: `candidate` | `confirmed` | `dismissed` | `archived`
- `obs_type` (optional) — filter by type: `oil_slick` | `chlorophyll_a` | `turbidity` | …

**Response 200:**
```json
{
  "items": [
    {
      "id": "obs-001",
      "analysis_run_id": "run-001",
      "scene_id": "S1A_IW_GRDH_...",
      "obs_type": "oil_slick",
      "evidence_class": "measured",
      "geometry": { "type": "Polygon", "coordinates": [[...]] },
      "area_km2": 5.0,
      "confidence": 0.85,
      "status": "confirmed",
      "created_at": "2024-02-07T12:00:00+00:00"
    }
  ],
  "count": 1
}
```

**INV-3 guarantee:** `evidence_class` is always one of `"measured"`, `"modeled"`, `"inferred"`.
For oil-slick observations from SAR imagery, `evidence_class` is always `"measured"`.

---

### `GET /aois/{aoi_id}/predictions`

List Predictions for an AOI. Responses include embedded ForecastFrames and an Open-Meteo
attribution field required by the CC BY 4.0 data licence.

**Response 200:**
```json
{
  "items": [
    {
      "id": "pred-001",
      "predictor_id": "oil_trajectory_v1",
      "kind": "trajectory",
      "evidence_class": "modeled",
      "uncertainty": { "particle_spread_km": 18.0 },
      "created_at": "2024-02-07T12:00:00+00:00",
      "frames": [
        {
          "id": "frame-001",
          "prediction_id": "pred-001",
          "valid_at": "2024-02-08T00:00:00+00:00",
          "footprint": { "type": "Polygon", "coordinates": [[...]] },
          "particle_count": 1000,
          "stats": { "mean_lon": -61.25, "mean_lat": 11.15, "spread_km": 12.3 }
        }
      ]
    }
  ],
  "count": 1,
  "_attribution": "Weather data by Open-Meteo.com (CC BY 4.0)"
}
```

**INV-9 guarantee:** `uncertainty` is always present and non-empty. A predictor without
uncertainty quantification is not permitted in the production system.

**Attribution:** `_attribution` must be preserved in any UI or downstream system that
displays prediction data (Open-Meteo CC BY 4.0 requirement).

---

### `GET /aois/{aoi_id}/impact`

List ImpactAssessments for an AOI. Each assessment links a Prediction to an ExposureLayer
with an estimated time of arrival (ETA).

**Response 200:**
```json
{
  "items": [
    {
      "id": "ia-001",
      "prediction_id": "pred-001",
      "exposure_layer_id": "coast-tobago",
      "valid_at": "2024-02-08T00:00:00+00:00",
      "eta_hours": 24.0,
      "metrics": { "coast_length_km": 12.5 }
    }
  ],
  "count": 1,
  "_attribution": "Weather data by Open-Meteo.com (CC BY 4.0)"
}
```

**Attribution:** `_attribution` must be preserved — impact ETA is derived from
Open-Meteo-forced trajectory predictions (CC BY 4.0 licence).

## Schema Reference

All schemas are defined in `argus/api/schemas.py` and serialized with Pydantic v2.
The contract is locked by `tests/test_api_contracts.py` — any breaking change to a
required field will cause a contract test failure.

### Breaking-change policy

A change is **breaking** if it:
- Removes a required field from any response
- Changes the type of an existing field
- Changes the name of the `_attribution` field
- Removes the `_attribution` field from any Open-Meteo-sourced response

Non-breaking changes (safe to deploy without version bump):
- Adding optional fields with defaults
- Adding new endpoints
- Adding new query parameters with defaults

---

## Error Responses

All errors follow the FastAPI default format:

```json
{"detail": "<human-readable message>"}
```

| Status | Meaning |
|---|---|
| 404 | Resource not found (AOI ID doesn't exist) |
| 422 | Validation error (malformed AOI geometry) |
| 500 | Unexpected server error |

---

## Attribution Requirements

Any client or UI that displays data from the following endpoints **must** show the
Open-Meteo attribution string:

- `GET /aois/{id}/predictions` — `_attribution` field
- `GET /aois/{id}/impact` — `_attribution` field
